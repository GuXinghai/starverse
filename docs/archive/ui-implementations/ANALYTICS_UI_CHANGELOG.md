### 动画时长
```
快速: 150ms   (离场)
标准: 200ms   (交互)
舒缓: 300ms   (悬停)
缓慢: 500ms   (入场)
```

### 圆角规范
```
小: rounded-lg    (8px)
中: rounded-xl    (12px)
大: rounded-2xl   (16px)
圆: rounded-full  (50%)
```

### 阴影层级
```
shadow-sm   → shadow-xl    (KPI 悬停)
shadow-md   → shadow-xl    (图表悬停)
shadow-lg   → shadow-2xl   (编辑面板)
```


```css
/* 在 style 块中 */
```
